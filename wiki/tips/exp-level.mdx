# レベルアップに必要な経験値

## 概要
経験値を上げるために必要な経験値は以下である。  
ちなみに、**12w23a**のスナップショットにて、差分が変わるようになった。

## データ
| レベル | 経験値 |
| ------------ | ---- |
| 1~10 | 17n |
| 15 | 255 |
| 20 | 370 |
| 25 | 560 |
| 30 | 825 |

## 計算式
[かまどを使用した](/wiki/furnance-can)経験値集めの場合
```
1/アイテム1つの経験値*レベルに応じた経験値
```
で、レベルを上げるために必要とする精錬ブロック数を求めることができる。
